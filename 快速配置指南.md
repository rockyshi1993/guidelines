## ✅ 下一步操作

### 1️⃣ 打开全局配置文件

**在 VS Code 中**：
1. 按 `Ctrl+Shift+P`
2. 输入：`Preferences: Open User Settings (JSON)`
3. 按回车

---

### 2️⃣ 复制粘贴这段配置

```json
{
  "github.copilot.chat.codeGeneration.instructions": [
    {
      "text": "你必须严格遵守 guidelines/v3.md 中定义的所有规范"
    },
    {
      "text": "你必须按照 23个STEP 执行流程，在 STEP 7/10/12/14/16 等待用户确认"
    },
    {
      "text": "代码覆盖率必须≥80%，复杂度≤10，0敏感信息泄露"
    },
    {
      "text": "你必须生成完整的代码、测试、文档和Swagger"
    }
  ],
  "github.copilot.chat.localeOverride": "zh-CN"
}
```

**⚠️ 注意**：
- 如果你的配置文件已有其他内容，在末尾添加时记得加逗号
- 不要重复已有的配置项

---

### 3️⃣ 保存并重启

1. 保存文件：`Ctrl+S`
2. 重新加载窗口：`Ctrl+Shift+P` → 输入 "Reload Window" → 回车

---

### 4️⃣ 为新项目准备规范文件

**每次创建新项目时**，从这里复制规范文件：

```
从: D:\OneDrive\Project\common\guidelines\

复制这些到你的新项目根目录：
  ├── guidelines/
  │   └── v3.md
  └── specs/
      ├── core/
      ├── rules/
      └── output/
```

**PowerShell 命令示例**：
```powershell
# 假设你的新项目在 D:\MyProject
$source = "D:\OneDrive\Project\common\guidelines"
$target = "D:\MyProject"

Copy-Item "$source\guidelines" -Destination $target -Recurse
Copy-Item "$source\specs" -Destination $target -Recurse
```
