## 关键提示
1. **编码风格**: 4空格缩进，LF行尾，UTF-8编码，行宽≤100
2. **输入校验**: 进行严格的参数验证，使用合适的校验方案
3. **日志规范**: 仅输出形状/字段集合，不含敏感数据
4. **文档联动**: 修改 API 时必须同步更新 README.md 和 CHANGELOG.md
5. **测试覆盖**: 新增功能需提供测试用例和 examples/ 示例
